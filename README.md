# 厦门大学马来西亚分校算法竞赛（程序设计竞赛）兴趣小组

这是一个由厦门大学马来西亚分校学生自发组织的兴趣小组，致力于为我校学生在 ICPC / CCPC 的报名、注册队伍和训练等事务上提供力所能及的帮助。

如果你非常热爱算法竞赛并有一定基础，欢迎加入我们！

执行组长微信：zeyu10007，小组主要邮箱：[xmumcpc2025@outlook.com](mailto:xmumcpc2025@outlook.com)。
